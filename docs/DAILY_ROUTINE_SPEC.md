# Sadhana: Daily Routine Specification
## The Diurnal Operating Cycle v3.4

**Scope:** Time-based flows, prompts, and rituals across the 24-hour cycle
**Principle:** The app serves the body's rhythm, not the other way around
**Core Truth:** Consistency builds the vessel. The routine is the practice.

---

# PART I: THE DAILY ARC

## 1.1 Circadian Architecture

Sadhana divides the day into **five windows**, each with distinct biological purposes and available practices.

| Window | Time | Biological State | Primary Function |
|--------|------|------------------|------------------|
| **Brahma Muhurta** | 4:00-6:00 | Cortisol rising, veil thin | High-voltage practice (if eligible) |
| **Morning** | 6:00-11:00 | Activation phase | Capture dreams, set intentions, gentle ignition |
| **Day** | 11:00-17:00 | Peak cortisol, peak output | Transmutation, missions, deep work |
| **Evening** | 17:00-21:00 | Cortisol declining | Return to baseline, process, close loops |
| **Night** | 21:00-4:00 | Parasympathetic dominance | Sleep protection, no intensity, seal |

## 1.2 The Non-Negotiables

Every day, regardless of phase:

| Ritual | Timing | Duration | Purpose |
|--------|--------|----------|---------|
| **Morning Check-in** | Within 30 min of waking | 10-30s | Establish baseline |
| **Seal the Day** | Within 60 min of sleep | 60-90s | Close the container |

---

# PART II: MORNING PROTOCOL (6:00-11:00)

## Wake Sequence

1. **Sensor Snapshot (Silent)** - 2-6 seconds
2. **Mode Pre-set** - System determines mode before user interaction
3. **Morning Check-in Card** - Where are you now?
4. **Drift Check** - Compare mood to sensor data
5. **Route to Primary Action**

## Phase-Specific Morning Flows

- Phase 1: Check-in only (30s)
- Phase 2: Check-in + Dream Capture (3h window)
- Phase 3: Check-in + Dream + Mission Selection
- Phase 4+: Check-in + Dream + Dyad Status
- Phase 6-7: Brahma Muhurta HV option (4:00-6:00 only)

---

# PART III: DAY PROTOCOL (11:00-17:00)

## Deep Work Flow (Phase 3+)

1. Bellows Breath (90s)
2. Intention Set (15s)
3. Deep Work Timer (25:00)
4. Output Log (30s)
5. Return Check

## Blocked After 17:00
- New Deep Work sessions
- Bellows breath
- Neti/Serpent
- Transmute Now

---

# PART IV: EVENING PROTOCOL (17:00-21:00)

## Return to Baseline

- If Yellow → Evening Notice → Return to Green options
- If Red → Evening Alert → Heavy Earth + Churn → Seal
- Mission Resolution required before seal
- Dream Intention Setting (Phase 2+)

---

# PART V: NIGHT PROTOCOL (21:00-4:00)

## Seal the Day Flow

1. Day Review (summary)
2. Final Mood Plot
3. Gratitude (optional)
4. Seal Complete → Night Mode

## Night Mode

- Extreme dim UI
- No notifications
- Only Emergency Downshift available
- Red-light color scheme

---

# PART VI: NOTIFICATION PHILOSOPHY

**Max 4 notifications/day:**
- Morning check-in (30 min after typical wake)
- Dream capture window (Phase 2+, within 3h of wake)
- Mission reminder (1x/day max if accepted)
- Seal reminder (90 min before typical sleep)
